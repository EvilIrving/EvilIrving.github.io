# pnpm

useful command

```shell
pnpm init // init pnpm-lock.yaml
pnpm install // install packeage from package.json
pnpm add <package> // like npm install <package>,
pnpm remove <package> // like npm remove <package>
pnpm update <package> // like npm update <package>
pnpm add -g <package> // like npm install -g <package>


pnpx and  pnpm dlx and pnpm create

pnpm is temp install and run some package, clearing it after run command of package

pnpx is an alias for pnpm dlx

pnpm create // create a new package by start kit 
```
